Example code and screencast for integrating Tumblr OAuth into your PHP project, forked from Jacob Budin's Tumblr Oauth library and examples.

Source files: https://github.com/timgavin/tumblroauth
Screencast: http://youtu.be/xdK0dXluKIQ


- Upload the files to your server.
- Change the values in config.php and /includes/db.php.
- Add the users_oauth table to your MySQL database.
- Watch the screencast.
- Visit index.php or post.php to get things started!
